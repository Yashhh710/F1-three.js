# 🏎️ F1 Desk Drive 3D

A browser-based 3D Formula 1 driving simulator where you can drive an F1 car across a realistic desk environment filled with everyday objects. The project focuses on immersive visuals, smooth controls, cinematic cameras, and interactive physics.

![Live-demo](preview.png)

## ✨ Features

- 🏎️ Realistic 3D Formula 1 car
- 🎮 Smooth keyboard driving controls
- 🎥 Multiple camera modes
  - Third Person
  - First Person
  - Cinematic Camera
- 🌍 Detailed desk environment
- 💡 Dynamic lighting and shadows
- 🚗 Wheel steering and wheel rotation animation
- 💥 Physics-based object collisions
- 📦 Interactive props
- 🌊 Custom materials and shaders
- 📱 Responsive design
- ⚡ Optimized rendering for modern browsers

## 🎮 Controls

| Key | Action |
|------|--------|
| W / ↑ | Accelerate |
| S / ↓ | Brake / Reverse |
| A / ← | Turn Left |
| D / → | Turn Right |
| C | Change Camera |
| R | Reset Car |

## 🚀 Getting Started

1. Clone the repository

```bash
git clone https://github.com/yourusername/F1-Desk-Drive-3D.git
```

2. Open the project folder

```bash
cd F1-Desk-Drive-3D
```

3. Start a local server

Using VS Code Live Server or:

```bash
python -m http.server
```

4. Open

```
http://localhost:8000
```
